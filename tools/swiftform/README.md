# swiftform
**by nitmoi** · necessity is the mother of invention

**Status:** v1 built — branding to be applied
**Category:** Document tools
**Privacy:** All data processed locally — files never uploaded, never stored

---

## Origin
Built out of frustration at paying for Adobe Acrobat just to fill in a form. A single HTML file that does the job without the subscription.

---

## What it does
- Upload any PDF in the browser
- Add draggable text annotations anywhere on the document
- Draw a signature with mouse or touchscreen
- Drag and reposition both text and signature freely
- Export / print the completed document
- Everything runs locally — no server, no upload, no data retention
